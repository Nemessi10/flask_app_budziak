Про себе

Я — програміст із розумінням інженерії програмного забезпечення та досвідом роботи з різними технологіями. Моя мета — створювати інноваційні та ефективні програмні рішення, використовуючи найсучасніші методики і технології. Протягом останніх кількох років я працював над різноманітними проєктами, від створення веб-застосунків до розробки програм для обробки великих даних.

Освіта

Я прямо зараз навчаюся на 3 курсі спеціальності Інженерія програмного забезпечення. Протягом навчання я здобув теоретичні знання та практичні навички у різноманітних сферах програмування та розробки програмного забезпечення, включаючи розробку на Java, Python, а також роботу з базами даних та фреймворками, такими як Spring Boot та Flask.

Навички

  • Мови програмування: Python, Java, JavaScript, C++, Ruby
  • Фреймворки та технології: Spring Boot, Flask, Angular
  • Інструменти: Git, Docker, PostgreSQL, MySQL
  • Методології: REST API
  • Інші навички: HTML, CSS, розробка адаптивних інтерфейсів

Досвід роботи

На даний момент я працюю в команді PNUdev Fullstack Developer, де займаюсь розробкою та підтримкою веб-застосунка за допомогою Spring Boot. Також я розробляю користувацький інтерфейс за допомогою Angular.

Проєкти

Я завжди прагну до розвитку та покращення своїх навичок через роботу над різноманітними проєктами. Ось деякі з них:

  • Проєкт 1: Система управління записами з використанням Spring Boot та Angular для університету. Цей проєкт включав створення CRUD інтерфейсів, інтеграцію з базою даних MongpDB та реалізацію авторизації через Spring Security.
  • Проєкт 3: Веб-застосунок на Spring Boot для генерації меню зі страв для користувачів.
